📋 Protocolo: animal abandonado

1. Mantén la calma

No te acerques bruscamente si el animal está asustado o agresivo.

2. Evalúa la situación

Comprueba si está herido, deshidratado o en peligro inmediato (carretera, calor, etc.).

3. Seguridad primero

Si es peligroso, no intervengas directamente. Llama a emergencias o policía local.

4. Contacta ayuda

Llama a 112, protectoras locales o policía para activar rescate.

5. Si puedes ayudar

Ofrece agua y comida solo si el animal está tranquilo y seguro.

6. Localización

Envía ubicación exacta para facilitar el rescate.

7. Nunca lo abandones

Espera ayuda o contacta una protectora hasta que llegue asistencia.
